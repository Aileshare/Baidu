clientWidth--可见窗口

    clientWidth = width + padding - 滚动条
    clientHeight = height + padding - 滚动条  
   

offsetWidth--元素自身(ie自动扩展: +子孙内容撑出)

    offsetWidth = width + padding + border (包括滚动条)
    offsetHeight = height + padding  + border(包括滚动条)  
    CSS中的margin属性，与clientWidth、offsetWidth、clientHeight、offsetHeight均无关
   

scrollWidth--元素自身 (ie自动扩展: +子孙内容撑出)

    scrollWidth = width + padding + margin + border (包括滚动条)
    scrollHeight = height + padding + margin + border(包括滚动条)  
   
   

clientLeft

    clientLeft = 边框宽度
    clientTop = 边框宽度
   

offsetLeft

        FF:dom与其参照定位的offsetParent之前的距离,从dom的border开始到offsetParent的border(内)
        IE:dom与其参照定位的offsetParent之前的距离,从dom的border开始到offsetParent的border(外)
